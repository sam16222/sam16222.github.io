Gesture Recognition for Google Cardboard

Involved in the development of an Augmented Reality based inspection framework for Android devices(Google Glass, Google Cardboard, Tablets). As a part of that, I worked on a hand gesture based interface for Google Cardboard. The main idea was to limit the computational complexity to as low as possible, so as to make it a feasible AR utility for low to mid-range smartphones. The work proposed a novel hand gesture based user interaction method in FPV, which is explored for (i) suitability of working with any smartphone monocular camera, without any built-in depth sensors, (ii) large scale real-time de- ployments without any network dependency, and (iii) reliability and accuracy in dynamic outdoor settings. A paper based on this work has been accepted at IEEE WACV 2017. Following this, I worked on region of interest selection using hand gestures, from an AR wearable in applications built around tele-presence assistance, and conducted a feasibility study on multiple algorithms for egocentric real-time gesture recognition, including color based methods, cascading classifiers and Faster R-CNN.

BCI based system for gauging attention levels

The system is proposed as an assistant tool for distance learning, involving a two node BCI headset and a Kinect sensor. The data obtained from the sensors is marked and processed using the k-Nearest Neighbors algorithm to create a prediction model for user attention levels. The model is then incorporated into a plug-n-play browser plugin, where it pauses the video when the attention levels fall below a threshold, to suggest alternate resources for the topic covered in that section. Also, academicians involved with MOOCs can obtain real-time feedback in the form of brainwaves and realize the strengths and weaknesses of a particular video. A paper based on this has been presented in IEEE TENCON 2016, Singapore.

AngularJS based Data Delivery framework

As part of my internship at Morgan Stanley Advantage Services, I was in-charge of the development of a automated Client Interaction Data Delivery(CIDD) application framework as a one stop solution for enrichment, validation and submission of client interaction data. The application runs on RESTful web services, synchronised with multiple data warehouses using DB2, with an AngularJS based web interface.

VLIW Pipelined Architecture

As part of my Computer Architecture course, I designed a pipelined structure for a multi issue VLIW architecture with a write-through cache. This hardware was implementated in Verilog hardware description language. These two hardware units were then integrated into a single unit.

E-Wallet

As part of my course on Object Oriented Programming, I created a Java Spring based E-Wallet, running as a native application on windows machines. This application was well received by the university and is currently being modified for deployment across the campus.

Word Recommendation System

I designed a word recommendation system using the test data from 340000 webdocs, using the Apriori algorithm on transactional data in partial fulfilment of the Data Mining course. The system ran on a 4-node cluster using Hadoop.
